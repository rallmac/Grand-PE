# Grand-PE (Next.js 14)

Pixel-close ecommerce frontend using original assets.

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS

## Structure
- app/: pages (Home, Categories, Product, Cart)
- components/: reusable UI components
- lib/: utilities and cart state
- styles/: global styles
- data/: mock product and category data

## Run
```bash
npm install
npm run dev
```

## Notes
- Images use placehold.co placeholders with approximate aspect ratios.
- Fonts use Google Fonts (Inter for body, Poppins for display).
- Animations are subtle via Tailwind transitions.
